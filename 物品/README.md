## 常用配方参考

### 水质处理标准配方（12L水箱）

#### 去氯处理
- 焦亚硫酸钠：0.018g（用牙签尖蘸取）
- 操作：加入水中搅拌30秒

#### 基础营养配方（维持健康）
- EDDHA铁：0.06g（≈1/50茶匙）
- 腐殖酸：1.2g（绿豆大小颗粒）
- 柠檬酸：0.6g（约1/8茶匙）

#### 急救营养配方（出现黄叶）
- EDDHA铁：0.12g
- 腐殖酸：2.4g
- 柠檬酸：1.2g
